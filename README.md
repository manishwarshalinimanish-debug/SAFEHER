🚨 SafeHer – AI-Powered Women's Safety & Emergency Response Application 🛡️📍

Excited to share my latest Android development project, SafeHer, a smart women's safety application built with Kotlin, Jetpack Compose, and Material 3. The app is designed to provide rapid emergency assistance through real-time location sharing, intelligent safety features, and offline reliability.

✨ Key Features
🆘 One-Tap SOS Emergency Alert with a high-visibility emergency interface.

📍 Live GPS Location Tracking that continuously streams coordinates to emergency contacts during an active SOS.

💬 Native SMS Integration, allowing emergency messages to be sent through the phone's default messaging application with Google Maps location links.

🗺️ AI-Based Safest Route Planner that suggests safer routes by considering reported safety hotspots and historical risk zones.

⚠️ Community Hazard Reporting, enabling users to report unsafe locations that are instantly reflected on the safety map.

📹 Emergency Video Recording & Sharing, allowing users to capture evidence and share recordings with emergency contacts using Android's native sharing system.

📡 Offline SOS Caching, securely storing emergency alerts, contacts, and location data when there is no internet connection, then automatically preparing them for dispatch once connectivity is restored.

🎨 Modern Material 3 High-Density UI, featuring a clean dashboard, intuitive navigation, and responsive user experience.
